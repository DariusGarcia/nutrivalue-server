# nutrivalue-server

Express.js server for NutriValue web app client

# Installation

## Setting up the Express Server

#### To start the server, you can cd into the Server/ directory and run:

add env vars

```
PORT=4001
HOST='127.0.0.1'
USER='root'
PASSWORD=
DATABASE='nutriValue'
JWT_SECRET_KEY=
```

```
npm install
```

```
npm run start or nodemon index.js
```
